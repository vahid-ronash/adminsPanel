آموزش نصب

1 ایمپورت کردن کتابخانه
در برخی از موارد دانلود از سایت jcenter با اختلال روبرو می شود. در این حالت از روش اضافه کردن دستی به منظور ایمپورت کردن کتابخانه استفاده کنید.
روش 1: با استفاده از Gradle

  1. در build.gradle مربوط به اپلیکیشن وارد شوید. مراقب باشید اشتباها به فایل gradle مربوط به پروژه وارد نشوید.

  2. کد زیر را در dependencies اضافه کنید. 

    dependencies{
compile 'co.ronash.android:pushe-base:0.10.0'
compile 'com.google.android.gms:play-services-gcm:7.5.0'
          }
  3. در بالای صفحه سمت راست sync را بزنید تا گریدل سینک شود.
روش 2: اضافه کردن دستی
قرار دادن کتابخانه در اپلیکیشن
  1. کتابخانه پوشه را از این لینک دانلود کنید.
  2. به محیط اندروید استدیو وارد شوید و از مسیر File>New>New Module>Import .JAR or .AAR Package فایل دانلود شده را در پروژه خود وارد کنید.
اضافه کردن Dependencies

  1. در build.gradle مربوط به اپلیکیشن وارد شوید. مراقب باشید اشتباها به فایل gradle مربوط به پروژه وارد نشوید.

  2. کد روبرو را در gradle وارد کنید. دقت کنید در کد عبارت pushe-lib-0.10.0 باید با نام subproject name در هنگام وارد کردن ماژول در پروژه یکی باشد. 
    dependencies{
compile project (':pushe-lib-0.10.0')
compile 'com.google.android.gms:play-services-gcm:7.5.0'
        }
  3. در بالای صفحه سمت راست sync را بزنید تا گریدل سینک شود.
  4. در صورتی که با دانلود کتابخانه گوگل پلی سرویس با مشکل روبرو شدید، از این لینک فایل در اختیار گذاشته شده را دانلود و اکسترکت کنید و فولدر extras به همراه زیر فولدرهای آن را در فولدر sdk خود با آدرس زیر قرار دهید. برای پیدا کردن محل دقیق فولدر sdk خود می توانید آدرس آن را از داخل فایل local.propertie پیدا کنید. ...\AppData\Local\Android\sdk
2 تغییرات AndroidManifest.xml
  1. وارد پنل کاربری پوشه شوید و در قسمت اپلیکیشن ها، فایل مربوط به منیفست اپلیکیشن خود را دانلود کنید.
  2. فایل دانلود شده را در یک برنامه ادیتور باز کنید و محتویات آن را به AndroidManifest.xml پروژه خود اضافه کنید.

برخی از دسترسی ها (permissions) برای پوشه اختیاری است. این دسترسی ها برای ارسال هدفمند اعلان به کاربران است. شما قادر خواهید بود با استفاده از این دسترسی ها کاربران خود را دسته بندی کنید و اعلانهای خود را هدفمند ارسال کنید.

3 تغییرات MainActivity.java
  1. در قسمت import ، کد روبرو را وارد کنید. 
    import co.ronash.pushe.Pushe;
  2. در onCreate کد روبرو را وارد کنید. 
    Pushe.initialize(this,true);

    3. اگر در تابع بالا مقدار true گذاشته شود و دستگاه کاربر شما گوگل پلی سرویس نداشته باشد و یا آن را غیرفعال کرده باشد دیالوگی باز می شود و به کاربر پیشنهاد نصب گوگل پلی سرویس داده می شود تا قابلیت دریافت و ارسال پوش نوتیفیکیشن فعال شود. اگر در تابع بالا false را قرار دهید، این دیالوگ برای کاربر باز نمی شود.

4 تست و عیب یابی

پس از اینکه فایل کتابخانه را مطابق با آموزش داده شده نصب کردید، میتوانید پروژه را اجرا و بر روی دستگاه خود نصب و تست کنید.

  1. به صفحه نصب ها بروید.
  2. بعد از گذشت زمان کوتاهی یک ردیف مشخصات نصب به لیست نصب ها اضافه می شود که متعلق به گوشی شما است.
  3. یک اعلان تست ارسال کنید.دقت کنید تلفن همراه شما به اینترنت متصل باشد.
توصیه می شود اپلیکیشن را بر روی موبایل یا تلبت اجرا و تست کنید. اگر می خواهید اپلیکیشن را بر روی Emulator تست کنید، بر روی ایمولاتور شما باید Google API نصب باشد.
5 دریافت اطلاعات دلخواه JSON

شما میتوانید از طریق پنل برای اپلیکیشن خود اطلاعات دلخواه در قالب یک JSON ارسال کنید. برای اینکه بتوانید از این اطلاعات در اپلیکیشن خود استفاده کنید مراحل زیر را انجام دهید.

  1. کد روبرو را در AndroidManifest.xml زیر تگ <application> اضافه کنید. 
    <service
    android:name=".MyPushListener"
    android:exported="false">
    <intent-filter>
        <action android:name="co.ronash.pushe.RECEIVE" />
    </intent-filter>
</service>
  2. در پروژه خود یک class جدید ایجاد کنید و برای آن یک نام انتخاب کنید. پیشنهاد ما نام MyPushListener می باشد.
  3. کد روبرو را در آن قرار دهید. بر روی PusheListenerService و JSONObject که به رنگ قرمز در آمده اند رفته و کلید Alt+Enter را بزنید. 
    public class MyPushListener extends PusheListenerService {
                                @Override
    public void onMessageReceived(JSONObject message) {
                                // Your code
                                }
                                }
  4. از این طریق JSON دریافت میشود و با قرار دادن کد خود به جای your code می توانید از آن استفاده کنید.

4 اضافه کردن گوگل پلی سرویس
توجه داشته باشید اگر در اپلیکیشن خود برای استفاده دیگری از گوگل پلی سرویس استفاده می کنید، ریسورس های کتابخانه ای زیر ناقص است و برای مقاصد دیگر غیر از پوشه کارایی ندارد.

  1. دو کتابخانه google-play-services-base.jar و google-play-services-gcm.jar را از این لینک دانلود و اکسترکت کنید.

  2. بر روی File > Import کلیک کنید. Android > Existing Android Code into Workspace را انتخاب کنید و به محل دانلود کتابخانه های بالا بروید و هر دو کتابخانه را انتخاب کنید. تیک Copy projects into workspace را بزنید و کتابخانه ها را به پروژه اضافه کنید.
  3. این دو کتابخانه گوگل پلی را در Dependency های پروژه خود اضافه کنید. بر روی پروژه خود کلیک راست کنید و به شاخه Properties -> Android -> Library بروید و با فشردن کلید Add کتابخانه های google-play-services-base و کتابخانه های google-play-services-gcm را انتخاب کنید.
5 اضافه کردن android-support-v4.jar

یکی از مقدمات مهم برای صحیح اجرا شدن کتابخانه پوشه، وجود داشتن و قدیمی نبودن فایل android-support-v4.jar می باشد. برای به روز رسانی این فایل ا دانلود کنید و آن را جایگزین فایل android-support-v4.jar پروژه خود کنید. برای این کار به Workspace خود بروید و وارد فولدر پروژه و سپس وارد فولدر libs شوید و این فایل را جایگزین کنید و اگر این فایل از قبل موجود نیست آن را در این فولدر قرار دهید.

2 اضافه کردن ریسورس ها

برای افزودن فایلهای Resource کتابخانه پوشه و کتابخانه گوگل پلی سرویس، کدهای زیر را در کلاس Main پروژه خود و در بخش Project Attributes وارد نمایید: 

#AdditionalRes: < google play service-base library res folder path > , com.google.android.gms
#AdditionalRes: < library res folder path >, co.ronash.pushe

به جای < play service library res folder path > آدرس فولدر res داخل فولدر اکسترکت شده Google-Play-Services را آدرس دهی کنید. به جای < library res folder path& > آدرس فولدر res داخل فولدر اکسترکت شده کتابخانه پوشه را آدرس دهی کنید.

به طور مثال اگر دو فولدر اکسترکت شده را در دسکتاپ قرار داده باشید با توجه به آدرس دسکتاپ از آدرس دهی زیر استفاده کنید: 

#AdditionalRes: C:\Users\Your-UserName\DesktopGoogle play service-low size\google-play-services-base\res , com.google.android.gms
#AdditionalRes: C:\Users\Your-UserName\Desktop\pushe-0.10.0\reFs , co.ronash.pushe

5 اضافه کردن رسیور دریافت JSON دلخواه
شما میتوانید از طریق پنل برای اپلیکیشن خود اطلاعات دلخواه در قالب یک JSON ارسال کنید. برای اینکه بتوانید از این اطلاعات در اپلیکیشن خود استفاده کنید مراحل زیر را انجام دهید.
  1. برای دریافت JSON شما باید یک receiver به پروژه خود اضافه کنید. (این فایل در فولدر ملزومات که دانلود کردید قرار دارد)
  2. از مسیر project>add existing module در نرم افزار B4A ، ماژول PusheJsonService.bas را انتخاب کنید.
  3. کد زیر را در منیفست خود اضافه کنید: 
    AddManifestText(
<permission android:name="$PACKAGE$.permission.C2D_MESSAGE" android:protectionLevel="signature" />)
AddPermission($PACKAGE$.permission.C2D_MESSAGE)
AddPermission(com.google.android.c2dm.permission.RECEIVE)
' Push Service Receiver Attribute
SetReceiverAttribute(PusheJsonService, android:permission, "com.google.android.c2dm.permission.SEND")
' Service Receiver Text
AddReceiverText(PusheJsonService,
<intent-filter>
<action android:name="com.google.android.c2dm.intent.RECEIVE" />
<category android:name="$PACKAGE$" />
</intent-filter>
<intent-filter>
<action android:name="com.google.android.c2dm.intent.REGISTRATION" />
<category android:name="$PACKAGE$" />
</intent-filter>)
    4. از این طریق JSON دریاقت می شود و با قرار دادن کد خود به جای your code در ماژول PusheJsonService.bas می توانید از اطلاعات دریافت شده استفاده کنید.
